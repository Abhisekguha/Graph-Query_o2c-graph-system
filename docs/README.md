# Documentation Assets

This folder contains documentation assets for the SAP O2C Graph System.

## Screenshot Instructions

**To add the system screenshot:**

1. Take a screenshot of the running application showing:
   - 3D graph visualization with colored nodes
   - Query assistant panel with example conversation
   - Statistics and controls visible
   - Clear view of the architecture

2. Save the screenshot as `screenshot.png` in this folder

3. The main README.md will automatically display it at:
   ```
   ![SAP O2C Graph System - 3D Visualization](docs/screenshot.png)
   ```

## Recommended Screenshot Settings

- **Resolution**: 1920x1080 or higher
- **Format**: PNG (for better quality) or JPG
- **Size**: Keep under 2MB for fast loading
- **Content**: Show split view (graph + chat) for best demonstration
